# Documentación y Recursos para el ICPC

¡Bienvenidos al repositorio de preparación para el ICPC!

## ¿Qué es el ICPC?

El **ICPC** (International Collegiate Programming Contest) es la competición de programación competitiva a nivel universitario más antigua, grande y prestigiosa del mundo. En este certamen, equipos de tres estudiantes compiten para resolver la mayor cantidad de problemas algorítmicos y matemáticos complejos en un tiempo limitado (generalmente 5 horas), utilizando un solo ordenador. 

La competición fomenta el trabajo en equipo, la resolución de problemas bajo presión y un profundo conocimiento en estructuras de datos y algoritmos.

Como tu entrenador (coach), he recopilado en este repositorio un conjunto de problemas de competiciones recientes, libros y referencias para que podamos entrenar, discutir estrategias y mejorar nuestras habilidades.

### Problemas de Competición (`docs/`)

Estos archivos contienen conjuntos de problemas reales de diferentes competiciones para que practiquemos.

*   [`docs/aquecimento_en.pdf`](docs/aquecimento_en.pdf)
    *   **Descripción:** Problemas de la sesión de calentamiento (Warmup) de la *Maratona SBC de Programação 2024*.
    *   **Contenido:** Un set corto de 4 problemas (Detour, Amusement Park Adventure, K for More K for Less, Fatigue-Fighting Vacation) ideal para entrar en calor y probar el entorno.

*   [`docs/maratona_en.pdf`](docs/maratona_en.pdf)
    *   **Descripción:** Problem set oficial de la *Maratona SBC de Programação 2024*.
    *   **Contenido:** Contiene 12 problemas (de la A a la L) de diversos niveles de dificultad, utilizados en competiciones simultáneas en Latinoamérica.

*   [`docs/gpmx_2024_segunda_v1.4.pdf`](docs/gpmx_2024_segunda_v1.4.pdf)
    *   **Descripción:** Problemas de la Segunda Fecha del *Gran Premio de México 2024*.
    *   **Contenido:** Un extenso conjunto de 13 problemas algorítmicos (de la A a la M) propuestos en el circuito de México, excelente para simular un concurso completo.

*   [`docs/gpmx_2025_tercera_v1.2.pdf`](docs/gpmx_2025_tercera_v1.2.pdf)
    *   **Descripción:** Problemas de la Tercera Fecha del *Gran Premio de México 2025*.
    *   **Contenido:** Conjunto de 12 problemas algorítmicos (de la A a la L) de la sesión de concurso.

### Libros (`books/`)

Material bibliográfico para estudiar y dominar las técnicas de programación competitiva.

*   [`books/Competitive Programmer’s Handbook.pdf`](books/Competitive Programmer’s Handbook.pdf)
    *   **Descripción:** *Competitive Programmer's Handbook*.
    *   **Contenido:** Un excelente libro introductorio para aprender y repasar algoritmos, matemáticas y estructuras de datos enfocados en la programación competitiva.

*   [`books/Comptetitive Programming Stenvencp1.pdf`](books/Comptetitive Programming Stenvencp1.pdf)
